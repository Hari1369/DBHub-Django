# DBHub-Django-
Bridges multiple databases into one system


# Project Overview :-
Multiple Project Optimizer is a Django-based backend system designed to manage multiple projects, multiple databases, and multiple users from a single platform.

It currently supports:
    1) PostgreSQL
    2) MySQL
    3) SQLite
The architecture is scalable and tenant-ready, allowing new databases and users to be added easily in the future.

# Prerequisites :-
Make sure you have the following installed:
    1) Python 3.10+
    2) Django 4+
    3) PostgreSQL
    4) MySQL
    5) SQLite (default with Python)
    6) Virtual Environment (recommended)
